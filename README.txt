# GreenCube

Projet de développement d'une application Android pour assurer une gestion d'un parc de "GreenCube". 
Projet en collaboration entre les étudiants de la Technique d'Informatique du Cégep de Sept-Îles 
et l'Institut de Technologie de Maintenance Industrielle (ITMI). 

Développeurs : Alex Boudreault, Maxyme Chiasson & William Leblanc.

Vous trouverez à la racine même du projet, trois fichiers :


- sources.TXT
- Guide de l'utilisateur.PDF
- Guide de l'administrateur.PDF

Le fichier source contient certaines sources utilisées pour la réalisation de ce programme.

Les fichiers sont respectivement les guides d'administrateur et d'utilisateur de l'application. 
Il est important de bien les consultés afin de bien comprendre comment fonctionne l'application.
